A beautiful Love Quotes Widget. Displays "Love Quotes" (in English) on your Sidebar.
All "Love Quotes" are taken from Top-Gamos.com ( the Best Greek Wedding Portal )

== Description ==

This plugin adds a beautiful widget with love quotes on your sidebar.

1. Supports Images
1. Easy Configuration
1. Perfect for websites ralated to Love, Wedding, Personal Relations etc.

Visit our Website for more: (http://www.seofixing.com/ "WP Fixes & Updates")

== Installation ==

Automatic Installation
1. Add New Plugin
2. Search Love Quotes Widget
3. Install
4. Activate
5. Add the 'SF LoveQuotes' Widget to the sidebar from the widget menu
6. Configure
7. Done


Manual Installation
1. Copy the directory 'SF-lovequotes-widget' to your WordPress '/wp-content/plugins' folder.
2. Activate the 'SF LoveQuotes' plugin in your WordPress admin 'Plugins'
2. Add the 'SF LoveQuotes' Widget to the sidebar from the widget menu
3. Configure
4. Done


== Frequently Asked Questions ==

= Is the Widget Responsive? =
Yes the widget is responsive.

= What is the Width field? =

The maximum image width. Enter a number without .px
ex. http://www.facebook.com/seofixing

= What if i check the Image Link Option? =

This option links your Widget image with the full size image
Useful if your template can open images in Lightbox
ex. If you are using the "Lightbox Plus Colorbox" Plugin

New Love Quotes added frequently.
